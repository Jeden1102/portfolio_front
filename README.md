# Portfolio

My personal portfolio webpage.

# Technologies used

- Nuxt3
- Taiwlind
- Supabase
- Typescript
- Vuelidate
- Lottie

## Installation

1. Build and run dev.

```bash
npm run install
npm run dev
```
2. Copy env.example to .env and insert supabase connection.

```bash
cp .env.example .env
```
